# Alejandro Mora — Portafolio Personal 🚀

Este repositorio contiene mi portafolio personal, una tarjeta digital minimalista diseñada para mostrar mi experiencia como desarrollador frontend y diseñador UI/UX.

![Tech Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JS-blue)

## 🎨 Sobre el Proyecto
El objetivo de esta página es presentar de manera estética y funcional mis habilidades técnicas, estadísticas de trabajo y proyectos más relevantes. El diseño es **totalmente responsivo**, adaptándose a dispositivos móviles y escritorio.

### Características principales:
* **Diseño Minimalista:** Estética limpia inspirada en interfaces modernas.
* **Sección de Habilidades:** Barras de progreso visuales para representar el dominio técnico.
* **Dashboard de Proyectos:** Tarjetas interactivas con efectos de hover.
* **Acceso Rápido:** Enlaces directos a contacto, LinkedIn y CV.

## 🛠️ Tecnologías utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Flexbox, CSS Grid y variables para un diseño fluido.
* **Google Fonts:** Tipografías del sistema para optimizar la velocidad de carga.
